# Final-Project-SC

=== SISTEM MANAJEMEN BUKU DI PERPUSTAKAAN ===

Cara menjalankan server 
  1. Menginstall node.js (ver 16+)
  2. Jalankan perintah npm install
  3. lalu npm start atau node index.js

Penjelasan endpoint yang ada disini
   1. - createBook = function ini untuk menambahkan sebuah data buku dalam perpustkaan
   2. - showAllBooks = function ini untuk menampilkan seluruh data buku yang ada di perpustakaan
   3. - showAllBookByGenre = function ini untuk menampilkan buku sesuai dengan genre yang kita inginkan
   4. - showBookId    = function ini untuk menampilkan data buku akan tetapi pencarian sesuai id buku
   5. - updateBook = function ini untuk mengubah data buku yang ada di perpustakaan 
   6. - deleteBook = function ini untuk menghapus data buku yang sudah tidak terpakai di perpustakaan
